# 音轨编辑器功能
## 需求分析
用户需要一个可视化音轨编辑器：
* 波形显示，可手动调整分割位置
* 支持多条音轨（原音、修复后）
* 对每个片段进行识别、修复
* 最终拼接导出完整音频
## 技术方案
### 前端
* 使用 **wavesurfer.js** 显示波形和可拖动区域
* 自定义多音轨组件
* 片段列表管理
### 后端
* 提供波形数据 API（可选，wavesurfer 可直接加载音频）
* 音频切分 API
* 片段识别 API（已有）
* 片段修复 API（已有）
* 音频拼接/导出 API
## 实现步骤
### 1. 安装依赖
前端安装 wavesurfer.js 和 regions 插件
### 2. 创建音轨编辑器组件
`frontend/src/components/TrackEditor.tsx`
* 波形显示
* 可拖动的分割标记
* 缩放/滚动控制
### 3. 创建片段管理组件  
`frontend/src/components/SegmentList.tsx`
* 显示所有片段
* 每个片段：试听、识别、编辑文本、修复、替换
### 4. 创建多音轨视图
* 原始音轨
* 修复后音轨（修复的片段替换原片段）
### 5. 后端拼接 API
`backend/routers/lyric_fix.py` 添加：
* `POST /api/lyric-fix/merge-segments` - 合并片段导出
### 6. 整合到新页面
`frontend/src/pages/TrackEditorPage.tsx`
